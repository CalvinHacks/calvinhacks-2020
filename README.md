# CalvinHacks Website

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

CalvinHacks Event Website for 2020.

## Installation

```
git clone git@github.com:calvinhacks/calvinhacks-2020
npm i
```

## Usage

### Development server

```bash
npm start
```

### Production build

```bash
npm run build
```
